# HI

The State of Hawaii does not publicly publish a single, comprehensive document, or set of documents, that lists every NIST 800-53 control ID alongside the specific "organizationally defined parameters."

Hawaii delegates these specific values to internal System Security Plans (SSPs) and configuration baselines managed by the Office of Enterprise Technology Services (ETS) within the Department of Accounting and General Services (DAGS).
