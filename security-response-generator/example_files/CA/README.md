# CA

The state of California does maintain a centralized document listing the specific "organizationally defined parameters" for NIST 800-53 controls, but it is classified as confidential and not publicly available.
The document is usually referenced as SIMM 5300-A.  The simm-5300 link below provides instructions on how to get access to their documentation.
